
Le fichier "sites.json" est utilisé par l'extension SecurIA, il permet de gérer la liste des sites IA sans avoir à mettre à jour toute l'extension dans le store.

Lien de l'extension : https://microsoftedge.microsoft.com/addons/detail/sensia-avertissement-do/lohmiiebbnkdlmbblddeddeehebfdoim

Fonctionnement de l'extension :

Lorsqu'un utilisateur visite un site d'IA (ChatGPT, Claude, Gemini, etc.), l'extension affiche un popup de sensibilisation rappelant de ne pas partager de données sensibles, personnelles ou confidentielles.

Déploiement :

L'extension peut être déployée par GPO ou Microsoft Intune. Les utilisateurs ne peuvent pas la désactiver ou la désinstaller.

Sécurité :

Aucune donnée utilisateur n'est collectée ou transmise
L'extension ne lit pas le contenu des pages ni les saisies des utilisateurs
Le fichier sites.json contient uniquement des données de configuration
Toutes les entrées sont validées avant utilisation

Langue de l'extension : Français
